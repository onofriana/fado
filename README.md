## Upload das imagens para GitHub

1. No repositório GitHub, aceder à pasta `images/`
2. Clicar em **Add file → Upload files**
3. Arrastar os 4 ficheiros `.jpg` desta pasta
4. Commit com mensagem: `Add performance photos`
